Get access to beta versions

You can install the latest beta version of irplus by using the following link.

irplus - Beta version (Integrated IR blaster)

irplusWAVE - Beta version (Homebrew IR blaster)

Notes

If you encounter a crash or issue please report it via the report button and/or via mail. You can always switch back to stable if you encounter an issue with beta.

There isn't always a beta version available. If there is no beta version and you are a tester, you will simply get the stable (production) one.
